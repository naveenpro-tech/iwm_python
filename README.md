# Movie Madders Codebase

This codebase has been reorganized into the following structure:

- **`/frontend`**: Contains the Next.js frontend application.
- **`/backend`**: Contains the Python FastAPI backend application.
- **`/archive`**: Contains documentation, old scripts, reports, and unused code.

## Getting Started

### Frontend
Navigate to the `frontend` directory:
```bash
cd frontend
npm install
npm run dev
```

### Backend
Navigate to the `backend` directory:
```bash
cd backend
start.bat
```
